# Social Media Platform - Django

A mini social media platform built using Django.

Features:

* User Registration and Login
* Create Posts
* View Personal Posts
* Logout Functionality
* Delete Account Option
* User Authentication

Tech Stack:

* Python
* Django
* HTML
* CSS
* SQLite

How to Run:

1. Install dependencies
2. Run migrations
3. Start server

Commands:

python manage.py migrate

python manage.py runserver
